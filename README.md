# Liferay E-Services Portal

## Project Overview

This project is an **E-Services Portal** developed using **Liferay** that allows users to apply for government services online and track the progress of their applications.

The portal provides a simple and user-friendly interface for citizens to:

* View available services
* Apply for services online
* Upload required documents
* Track application status using tracking ID

---

# Features

## 1. Service Listing

The portal displays multiple services in card format.

Available services include:

* TIN Number for Micro Enterprise
* TIN Number for Individual
* TIN Number for Large Enterprises
* Trade License for Individual

Each service card contains:

* Department
* Processing Time
* Service Fee
* Required Documents
* **Learn More**
* **Apply Now**

---

## 2. Service Details (Learn More Page)

Users can view detailed information about the service including:

* Department responsible for the service
* Processing time
* Service fee
* List of required documents

Example required documents:

* Fayda Passport
* House Map
* Rental Agreement / Land Ownership
* Valid Identification

---

## 3. Application Tracking

Users can track all their submitted applications using the **Track Your Applications** page.

The tracking system displays:

* Tracking ID
* Service Name
* Department
* Status
* Submission Date

Users can also search applications using:

* Tracking ID
* Service Name
* Status filter

---

## 4. Online Application Submission

Users can apply for services by:

1. Selecting a service
2. Filling out the application form
3. Uploading required documents
4. Submitting the request

After submission, the system generates a **Tracking ID** for monitoring the application.

# System Workflow

1. User logs into the portal
2. User selects a service
3. User views service details
4. User submits the application
5. System stores the application data
6. User tracks the application status
